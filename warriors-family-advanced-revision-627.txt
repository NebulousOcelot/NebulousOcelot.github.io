  + If you're unsure about cat colors, have a look at <a href\="https://cedarseed.com/portfolio/housecat-colors/" target\="_blank">this chart</a>. 
  + Not all genes may be accurate - some are missing (colorpoint and silver), some have been split into two loci (Wd/Ws). The wild type allele for curly/hairless is currently represented by Se. 
  + In this generator, each descendant has an eye color within the range of their parents' eye colors (in order: copper, amber, gold, yellow, hazel, green). Well, mostly - sometimes the main cat's eye color is not within the range. I do not have a fix for that.
  + Blue eyes only occur if the cat is white (Wd allele present) or has white spotting (Ws allele present), and it's not a guaranteed chance.
  + If you generate a cat with the -leaf suffix, that cat is a medicine cat (you are also free to ignore this; this is merely an inherited bit of fanon from cat-generators (linked below)).
  + "Older" X or "Younger" X means that the cat belongs to a different litter. Cats can have up to three litters here (four, if you've got a Queen).
  + If you want to generate a cat's family, but that cat already has a mate and children, use "Mate" or "Elder's Mate" mode.
  + If you're looking to generate grandparents, aunts, uncles, and cousins, generate the main cat's parents first, then input each parent as the main cat under "Mate" or "Elder's Mate" mode.
